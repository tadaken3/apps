# Ken.T Portfolio - AXIS

> 1%良い選択を、毎日に。

AXISテンプレートをベースにした、個人ポートフォリオサイトです。
ブランドストーリー、開発したiOSアプリのギャラリーを掲載しています。

## 🌐 Site URL
[https://tadaken3.github.io/apps](https://tadaken3.github.io/apps)

## ✨ Features
- **AXIS Template**: 大胆なタイポグラフィとグリッドレイアウトを採用したモダンなデザイン
- **Brand Story**: 「人生は、一発逆転で変わらない。」から始まるブランドストーリー
- **Responsive Layout**: デスクトップ・モバイル両対応
- **App Gallery**: アプリのスクリーンショットで視覚的にアピール
- **Marquee Animation**: 流れるテキストによる動的な演出
- **PWA Ready**: ホーム画面に追加してアプリのように利用可能

## 🛠 Technologies
- HTML5
- CSS3 (CSS Variables, Animation, Flexbox/Grid)
- JavaScript (Vanilla JS)
- FontAwesome (Icons)
- Google Fonts (Inter, Noto Sans JP)

## 📱 Apps Showcase
- **旬で選ぶ買い物メモ**: 旬の食材がすぐにわかる買い物メモアプリ
- **Your Time**: 寿命を見える化し、目標に集中するアプリ
- **Fit Routine**: 運動習慣を継続しやすくする記録アプリ
- **超集中タイマー**: ポモドーロ・テクニックを活用した集中タイマー
