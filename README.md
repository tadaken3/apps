# Portfolio - GLASS

グラスモーフィズム（すりガラス）デザインを採用した、個人のポートフォリオサイトです。
自己紹介、SNSリンク、開発したiOSアプリのギャラリーを掲載しています。

## 🌐 Site URL
[https://tadaken3.github.io/apps](https://tadaken3.github.io/apps)

## ✨ Features
- **Glassmorphism Design**: 美しいすりガラス効果を取り入れたモダンなUI
- **Responsive Layout**: デスクトップ・モバイル両対応
- **Compact Profile**: アプリ一覧（Works）を際立たせるコンパクトなプロフィール設計
- **App Gallery**: 開発したアプリの特徴を画像と説明文で紹介

## 🛠 Technologies
- HTML5
- CSS3 (Vanilla CSS, Animation, Flexbox/Grid)
- JavaScript (Vanilla JS for dynamic content rendering)
- FontAwesome (Icons)
- Google Fonts (Quicksand, M PLUS Rounded 1c)

## 📱 Apps Showcase
- **Your Time**: 寿命を見える化し、目標に集中するアプリ
- **超集中タイマー**: ポモドーロ・テクニックを活用した集中タイマー
- **Fit Routine**: 運動習慣を継続しやすくする記録アプリ
